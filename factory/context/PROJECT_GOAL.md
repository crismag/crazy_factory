# Project Goal

## Primary Goal

Create Crazy Factory as a local-first autonomous software development apprentice that can make periodic, incremental, safe, traceable progress on software projects under owner oversight.

## Required Qualities

- local-first operation
- bounded autonomy
- specialized worker responsibilities
- transparent lifecycle phases
- durable memory across sessions
- backlog-driven task selection
- repository safety
- evidence-based reporting
- human approval for consequential actions

## Current Goal

The current goal is to establish the documentation operating system only. No implementation artifacts are authorized in this phase.
