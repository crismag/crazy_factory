# Target State

## Long-Term Target

Crazy Factory operates as a local-first autonomous software development apprentice capable of periodically selecting bounded work, making incremental progress, validating results, recording decisions, and preserving memory while remaining observable and owner-controlled.

## Target Capabilities

- goal understanding and architecture expansion
- backlog management and task planning
- future code and test generation
- independent review
- local git checkpointing
- reporting and memory updates
- recovery and stall detection
- local-model operation through Ollama
- optional scheduling and controlled integrations
- optional oversight and multi-model collaboration
- isolated multi-project operation

## Target Constraints

The final system must remain approval-aware, local-first, traceable, maintainable, and conservative around external or destructive actions.
